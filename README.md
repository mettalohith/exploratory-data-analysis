## 📖 Project Overview

This project performs an **Exploratory Data Analysis (EDA)** on a dataset containing **India's largest companies**. The dataset was **scraped from Wikipedia** and analyzed using Python to extract meaningful business insights, identify trends, and visualize company performance across different industries.

The notebook demonstrates a complete EDA workflow—from importing raw data to cleaning, preprocessing, analyzing, and visualizing the dataset.

---

## 🎯 Objectives

- Import and inspect the dataset
- Clean and preprocess raw data
- Handle missing values
- Perform statistical analysis
- Explore industry-wise distributions
- Visualize important business metrics
- Generate meaningful insights from the data

---

## 📂 Dataset

**Source**

The dataset was scraped from the official Wikipedia page:

🔗 **https://en.wikipedia.org/wiki/List_of_largest_companies_in_India**

The dataset contains information such as:

- Company Name
- Industry
- Revenue
- Profit
- Assets
- Market Value
- Employee Count
- Headquarters
- Other publicly available business information

> **Disclaimer:** The dataset is derived from publicly available information on Wikipedia and is used solely for educational and analytical purposes.
